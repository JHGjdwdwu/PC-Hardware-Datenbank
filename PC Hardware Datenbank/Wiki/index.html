Datenbank installation:
Zuerst sollte man die Datenbank Installieren dann die Software.


Programm installation:
Nach der Installation kommt gleich ein Hinweis das man die Datenbank angeben soll hier kann man wählen welche Datenbank man verwenden möchte. Dazu wird die IP-Adresse des Servers angeben der die Datenbank hostet sowie der Name der Datenbank der lautet pc-hardware-datenbank wen man keine eigene modifizierte Datenbank verwendet. Der Benutzername und das Password werden vom Administrator bestimmt, der den Datenbank Server installiert hier den Administrator fragen. Nachdem man die Daten gespeichert hatte kommt eine Bestätigung das die Daten erfolgreich gespeichert worden sind und wir können das Fenster schließen.



Erster-Login:

Beim ersten Login gibt es nur einen Benutzer, mit dem man nun Benutzer erstellen muss, bitte den Standard Benutzer ersetzen durch einen eigenen der nicht jedem bekannt ist.
Achtung hier kann man sich ausschließen!!! Bitte immer einen Administrator Account behalten, mit dem man sich wieder einloggen kann.
Benuzer: root
Password: sudo
Als Administrator kommen wir automatisch in das Administrationsmenü wo wir die Datenbank bestimmen und Benutzer verwalten können. Des Weiteren können wir auch Dateien eingeben oder Daten, die bereits eingegeben worden sind, betrachten.

Benutzer verwalten:
Zunächst sollte man den Administrator ändern dazu klickt man doppelt in der Tabelle den Benutzer root an. Nun sollten die Felder oben ausgefüllt sein die man bearbeiten kann hier ändert man den Namen und Password. Danach lägt man Benutzer an, die mit der Datenbank arbeiten sollen bei den Rechten hat man die Möglichkeit von:

lessen
Die eigegebenen Daten betrachten aber nicht ändern oder löschen.
schreiben
Neue Daten eingeben und daraus einen QR-Coder drucken. Bereits eingegebene Daten können nicht verändert werden.
lessen+löschen
Damit kann man Daten betrachten und auch löschen aber in der Version 1.0.0 noch nicht bearbeiten.
lessen+schreiben
So kann man neue Daten eingeben und daraus einen QR-Coder drucken. Auch kann man bereits eingegebene Daten betrachten aber nicht bearbeiten oder löschen.
lessen+löschen+schreiben
Man kann Daten eingeben QR-Coder daraus drucken, Daten betrachten und auch löschen. Was man mit der Version 1.0.0 nicht kann ist Daten zu bearbeiten.
root
Alls root kann man alles was die anderen Rechte hergeben und die Datenbank bestimmen. Auch kann man Benutzer verwalten und so Rechte vergeben.
Hinweis:
Das vergebene Password wird verschlüsselt in der Datenbank gespeichert und so auch im Feld unkenntlich angezeigt damit die länge nicht bestimmt werden kann!
Dateien eingeben:
Wen wir Daten eingeben wollen wählen wir im Menü zunächst die Art der Hardware zum Beispiel eine Grafikkarte. Dieses Menü können wir sehr flexibel auf dem Desktop platzieren so dass wir es als Leiste an der Seite platzieren können wen man mehr eingeben möchte. So dass man Festpatten und Grafikkarten eingeben kann ohne, dass man immer wieder Fenster schließen offen muss.

Wenn man nun einen Hardwaretype gewählt hat ersehnt ein individuelles Formular für die Hardware. Einige Felder besitzen eine Unterstützung für die Eingabe von Daten so das man Vorschläge bekommt. Hat man alles ausgefüllt kann man die Daten speichern und man wird gefragt ob man einen QR-Code drucken möchte. Alternativ kann man auch über die Schaltfläche einen QR- Coder drucken.
Hinweis:
In der Version 1.0.0 ist das die letzte Möglichkeit einen QR-Coder zu erzeugen!

Daten-Betrachten:
Das ist in der Version 1.0.0 noch sehr eingeschränkt und man kann hier nur Listen betrachten was in der jeweiligen Kategorie eingetragen worden ist. Eine detaillierte suche ist leider nicht möglich auch das Ändern von Daten ist nicht möglich. Man muss einen Eintrag neu schreiben und den alten löschen wen man eine Änderung Vornamen möchte.
